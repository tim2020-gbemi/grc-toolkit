# GRC Toolkit

Security automation scripts for GRC engineers and compliance analysts.

## compliance_mapper.py

Maps NIST CSF 2.0 controls to ISO 27001:2022 and SOC 2 TSC equivalents.

**What it does:**
- Cross-references controls across three major frameworks
- Tracks assessment status per control
- Exports audit-ready CSV reports

**Run it:**
python compliance_mapper.py

**Frameworks covered:** NIST CSF 2.0, ISO 27001:2022, SOC 2 TSC
